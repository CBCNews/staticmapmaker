You can copy and paste the code below into an HTML document or in CSS.


HTML


<img src="{{ mapcode }}" alt="Google Map of [[base.location]]">


CSS


  {
    background-image: url({{ mapcode }});
  }











You can add a Google static map to any website with simple HTML and no JavaScript. Using URL parameters you can describe to Google the type of map that you want and Google will send it back.


The Static Map Maker helps you configure the parameters and gives you the code in real time.


If your site receives high traffic, you might need an API key. According to Google, “Most website and applications may use the Google Static Maps API free of charge.” Read more about usage limits.


Check out the Static Maps API for more information.
